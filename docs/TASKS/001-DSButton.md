# TAREA 001 - DSButton

## Objetivo

Crear el primer componente oficial del DS Design System.

## Descripción

DSButton será el botón reutilizable que utilizará toda la plataforma.

No se usarán botones HTML directamente dentro del proyecto.

## Debe permitir

- Texto
- Iconos (más adelante)
- Estado deshabilitado
- Estado de carga (más adelante)
- Variantes:
  - Primary
  - Secondary
  - Outline
  - Ghost

## Debe ser

- Responsive
- Accesible
- Reutilizable
- Tipado con TypeScript
