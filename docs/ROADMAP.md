# MVP

El MVP estará terminado cuando el sistema permita:

- Mostrar productos
- Buscar productos
- Mostrar variantes
- Abrir galería de imágenes
- Contactar por WhatsApp
- Adaptarse a dispositivos móviles
- Configurar un negocio sin modificar código
